# Mobile development with B4X

This guide is a work in progress. It reflects my experience in developing real world Android apps with B4A.

B4X is a suite of simple, powerful and modern development tools from [Anywhere Software](https://www.idevaffiliate.com/33168/26.html) and it consists from: B4A, B4J, B4i, B4R.

_In this guide our main focus will be **B4A** but, we are also going to refer to B4i and B4J._
